# CRM作业助手

Chrome / Edge Manifest V3 扩展，用于在深空 CRM 工作台汇总当前教师在读营期中的旷课、课后作业未完成、课后拓展未完成和调课学员。

当前版本：`1.0.3`

版本号使用三段式十进制递增，每一位达到 `10` 时向前一位进位。

## 数据来源

- 常驻班级：自动读取“用户学情”的只读接口 `/live/learning-situation/searchLearningSituationList`。
- 本次上课班级、到课和作业：读取“教学期”的只读接口 `/live/class-user/teachSearch`。
- 课次数据更新时间：读取 `/live/class-user/teachTotal` 返回的课次专属 `updateTime`。
- 课次与班级时段：读取营期课程和班级信息接口。
- 匹配主键：CRM 学员 `userId`，不使用姓名推断。
- Excel/CSV 仅作为历史班级或时段修正，不再是使用扩展的前置条件。

扩展不会保存 CRM 密码或登录令牌，不调用 CRM 写接口。常驻班级修正、永久转班覆盖和按钮位置只保存在当前浏览器的 `chrome.storage.local` 中。

## 安装

1. 在 Chrome 打开 `chrome://extensions`，或在 Edge 打开 `edge://extensions`。
2. 开启“开发者模式”。
3. 选择“加载已解压的扩展程序”。
4. 选择本目录 `crm-learning-alert-extension`。
5. 重新加载 `https://sk-crm.codemao.cn/workbar`，等待 CRM 表格出现。
6. 点击页面右侧“作业统计”。

首次加载扩展后必须刷新 CRM，使只读请求捕获脚本在页面接口请求之前启动。

## 判定规则

- 旷课：教学期“是否到课”为空或未到课，或者“是否完课”为 `0%`；任一条件成立即判定。
- 课中作业未完成：课中作业任一已布置题型的通过数小于总数；可通过“课中作业”按钮独立筛选。
- 作业未完成：读取教学期“课后作业”列，创作题、OJ 题或客观题中任一已布置题型的通过数小于总数；`0/0` 忽略。
- 拓展未完成：独立读取教学期“课后拓展”列，以通过数小于拓展题目总数判定；题目数量动态读取，不限制为两道。
- 调课：用户学情常驻班级与该课次教学期班级不同，且 CRM `adjustmentState` 表示调课或已完成。
- 待确认：两个班级不同，但 CRM 调课状态仍为正常。
- 数据更新时间或课次结束时间缺失时不生成旷课结论。
- 三级加载：首次打开只读取全部在读营期；选择营期后加载班级；选择班级后加载全部课节。
- 实时查询：营期、班级和课节下拉选择均直接请求 CRM 接口，不读取或写入作业数据缓存。
- 查询调度：班级信息、学情名单和课节目录并行读取；多课节及分页请求使用受控并发，减少等待时间且不缓存接口数据。
- 刷新按钮：保留当前营期、班级、已选课节、分类和姓名/ID关键词，重新请求基础数据及当前作业分类接口。
- 重置按钮：清空营期、班级、课节、分类和姓名/ID关键词，恢复到初始搜索状态。
- 课节选择：不会默认选择课节，可在复选下拉菜单中同时选择一节或多节课，点击“确定”后合并读取作业数据。
- “复制当前结果 ID”会复制当前班级、全部所选课节和异常类型范围内的去重学员 ID。
- “导出 Excel”会导出当前筛选后实际显示的 6 列作业明细，学员 ID 以文本格式保存。
- 多课节结果按课节分别保留明细行，并在学生名字后显示对应课节；列表按学员 ID 数值倒序分组，同一学员的课节按最新优先排列。
- 课中作业、课后作业和课后拓展使用独立表格列与筛选分类，分别显示每种题型的“通过数/总数”和完成进度。
- 作业分类按钮复用 CRM 教学期接口 `POST /live/class-user/teachSearch`：课中作业、课后作业、课后拓展分别传递 `classUserTeacherWechatQuicklyOperate` 值 `5`、`6`、`8`，列表直接展示接口返回的学员。
- 三个作业分类会对接口结果二次筛选：OJ 题按通过数判断，客观题仅按提交数判断；客观题全部提交即视为通过，创作题不参与该筛选。

## 测试

在 PowerShell 中运行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\tools\serve.ps1 -Port 8765
```

然后打开：

- `http://127.0.0.1:8765/tests/index.html`：规则与适配器测试。
- `http://127.0.0.1:8765/tests/mock.html`：桌面和窄屏界面预览。

接口字段变化时优先修改 `src/crm-adapter.js` 和 `src/page-bridge.js`，名单格式与异常规则无需改变。
